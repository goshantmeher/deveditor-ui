**Button** is the primary interactive component for actions.

- **Variants:** `default` | `destructive` | `outline` | `secondary` | `ghost` | `link`
- **Sizes:** `default` | `sm` | `lg` | `icon`
- **asChild:** When `true`, renders the child as the root element (e.g. link, Link). Uses Radix Slot.
- **unstyled:** When `true`, strips all default styles. Keeps accessibility and data attributes.
